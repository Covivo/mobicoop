Table: proposal
Table CommentsTable des propositions (offres et demandes) de covoiturage : - soit un passager qui demande un trajet - soit un conducteur qui propose un trajet Une offre de trajet est composé de 2 étapes ou plus : - une étape d'origine - une étape de destination - éventuellement des étapes intermédiaires L'étape d'origine est numérotée 0, les autres étapes sont numérotées incrémentalement. Une demande de trajet est initialement composée de 2 étapes : - une étape d'origine - une étape de destination L'étape d'origine est numérotée 0, l'étape de destination est numérotée 1. En cas de trajet multimodal, la proposition originale est enregistrée, et une proposition calculée est générée. Cette proposition peut être composée d'étapes supplémentaires utilisant d'autres modes de transport (piéton, bus, tram...). La proposition est liée à un ensemble de critères de sélection (demande) ou de restriction (offre) qui se trouvent dans la table criteria.
